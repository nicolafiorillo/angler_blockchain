# AnglerBlockchain

**A blockchain implementation in Elixir.**

This is for educational purposes and it is not ready to use in production.

Usage:
```elixir
mix deps.get, deps.compile, compile
iex -S mix
```

### Roadmap
The Angler Blockchain is a work in progress. An incomplete list of functionalities with no particular order:

- [] Technology layer
  - [x] Raw blockchain
  - [] HTTP interface
- [] Protocol/Coin layer
  - [x] Mempool
  - [x] Transactions
  - [] Consensus protocol
  - [] Byzantine fault tolerance
- [] Token layer
